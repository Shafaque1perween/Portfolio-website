 Portfolio Website

A personal portfolio website built using HTML, CSS, and JavaScript to showcase my skills, projects, and contact information. The website highlights my frontend development journey and provides an overview of my work.

## Features

* Hero Section
* About Me Section
* Skills Section
* Projects Showcase
* Contact Section
* Smooth Navigation
* Clean and Modern UI

## Technologies Used

* HTML5
* CSS3
* JavaScript
